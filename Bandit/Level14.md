# Bandit Level 13 -> Level 14

## Objetivo
The password for the next level is stored in /etc/bandit_pass/bandit14 and can only be read by user bandit14. For this level, you don’t get the next password, but you get a private SSH key that can be used to log into the next level. Look at the commands that logged you into previous bandit levels, and find out how to use the key for this level.

### Teoria
Normalmente, para entrar por SSH usas un usuario y una contraseña. Sin embargo, SSH permite el uso de un par de llaves (Pública y Privada).

La Llave Privada (que es el archivo que tienes ahora, probablemente llamado sshkey.private) es como tu firma digital personal. Nunca debe compartirse.

El parámetro clave aquí es el flag -i (Identity File).

Cuando ejecutas el comando con esta flag, le dices al cliente SSH: "No me preguntes contraseña, usa este archivo para demostrarle al servidor que soy yo".

El comando para saltar al siguiente nivel sería:
<pre>
scp -P 2220 bandit13@bandit.labs.overthewire.org:sshkey.private .
ssh -i sshkey.private bandit14@bandit.labs.overthewire.org -p 2220
<pre>
